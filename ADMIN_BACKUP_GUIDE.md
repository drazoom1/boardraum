- ✅ 모든 승인된 베타 테스터의 게임 데이터
- ✅ 보유 게임 목록

- ✅ 플레이 기록

- 각 유저당 **최대 3개 백업 유지**
- 오래된 백업은 자동 삭제
- 새로운 백업이 생성되면 가장 오래된 것부터 삭제

## 🔍 백업 확인하기

### Supabase 대시보드에서 확인:
1. Supabase Dashboard 접속
2. SQL Editor 클릭
3. 다음 쿼리 실행:

```sql
-- 최근 백업 20개 보기
SELECT 
  key as "백업 키",
  (value->>'created_at') as "백업 시간",
  (value->>'user_id') as "유저 ID",
  (value->>'game_count') as "게임 수"
FROM kv_store_0b7d3bae
WHERE key LIKE 'backup_%'
ORDER BY (value->>'created_at') DESC
LIMIT 20;
```

### 브라우저 콘솔에서 확인:
```javascript
// F12 → Console에서 실행
const token = localStorage.getItem('supabaseAccessToken');
fetch('https://YOUR_PROJECT.supabase.co/functions/v1/make-server-0b7d3bae/data/backups', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(r => r.json())
.then(d => console.log('📋 내 백업 목록:', d.backups));
```

## 🎯 자주 묻는 질문 (FAQ)

### Q: 백업이 실패하면?
**A:** 다시 클릭하면 됩니다. 기존 데이터는 안전합니다.

### Q: 백업은 어디에 저장되나요?
**A:** Supabase 데이터베이스 `kv_store_0b7d3bae` 테이블에 저장됩니다.

### Q: 백업을 복구하려면?
**A:** 개별 유저는 "데이터 백업" 메뉴에서 복구할 수 있습니다.

### Q: 몇 명까지 백업할 수 있나요?
**A:** 제한 없습니다! 모든 승인된 베타 테스터를 백업합니다.

### Q: 백업 용량은?
**A:** 유저당 평균 100KB 정도입니다. 100명이면 약 10MB입니다.
